<pre>
Maven
   Maven是一个项目构建和管理工具
   Pom.xml(Project object model)
       在执行task时，maven会去项目根目录下读取pom.xml获取需要的配置信息
       包含元素：
           1）groupId 项目创建组织的唯一标识符
           2) artifactId 项目基地址名
           3）version  项目版本
           5）dependencies 依赖，在dependency中添加具体依赖
           6）repositories 用来存储artifact(依赖文件)的的，一般都是某个仓库
